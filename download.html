Version 1.0

Released May 10, 2015

This is the very first version of A'Tuin, an open source IDE for the creation of images.
This interpreter takes its instructions and draws a picture built from it. Use these instructions to move vertically or horizontally a virtual cursor. If the cursor is down each move leaves a mark, if moving in position high nothing is written. The instruction also allows the cursor to turn counterclockwise by a given angle in degrees.
SYSTEM REQUIREMENTS
  • 512 MB RAM minimum, 1 GB RAM recommended
  • 1024x768 minimum screen resolution
  • Oracle JRE 1.8+ or OpenJDK 1.8+
INSTALLATION INSTRUCTIONS
Windows System
  • Copy the A'Tuin-*.tar.gz to the desired installation location

  • Execute ATuin-1.0.exe

  • More Details
Linux System
  • Copy the ATuin-*.tar.gz to the desired installation location
    (make sure you have rw permissions for that directory)

  • Unpack the ATuin-*.tar.gz using the following command:
    tar xfz ATuin-*.tar.gz

  • Run a'tuin.sh from the bin subdirectory
