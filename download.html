Version 1.0

Released May 10, 2015

This is the very first version of A'Tuin, an open source IDE for the creation of images.
This interpreter takes its instructions and draws a picture built from it. Use these instructions to move vertically or horizontally a virtual cursor. If the cursor is down each move leaves a mark, if moving in position high nothing is written. The instruction also allows the cursor to turn counterclockwise by a given angle in degrees.
